PADELANALYSIS - V-CLEAN FINAL UI V2

- geen linkerbalk meer
- naam-only flow
- ID alleen tussen haakjes
- match explorer:
  - Datum / Week / Periodeblok voorbereid
  - Partner / Opponent 1 / Opponent 2 / Poule labels verbeterd
